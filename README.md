# 🌟 Smart To-Do List Application

A simple and interactive command-line To-Do List application developed in Python as part of the **CodSoft Python Programming Internship**.

This application helps users organize and manage their daily tasks efficiently through an easy-to-use menu-driven interface.

---

## 🚀 Features

- ➕ Add New Tasks
- 📋 View All Tasks
- ✏️ Update Existing Tasks
- 🗑️ Delete Tasks
- 🔍 Search Tasks
- ✅ Mark Tasks as Completed
- 📊 View Task Statistics
- 💾 Automatically Save Tasks (JSON)
- 📂 Automatically Load Saved Tasks
- 🏷️ Set Task Priority (High, Medium, Low)
- 📅 Store Task Creation Date & Time
- 🎉 User-Friendly Menu Interface

---

## 🛠️ Technologies Used

- Python 3
- JSON
- OS Module
- Datetime Module
